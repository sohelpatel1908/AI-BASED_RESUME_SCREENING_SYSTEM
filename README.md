# **AI-Based Resume Screening System**
## **Description**
### An intelligent resume screening system using NLP and Machine Learning to automatically rank job candidates based on skill match, relevance, and job similarity.
---
## **Features:**
- Resume Parsing
- Skill Extraction
- Semantic Matching
- Ranking Engine
- Candidate Shortlisting
- Web Interface <ins>(Streamlit)</ins>
- Explainable AI
---
## **Tech Stack:**
- Python
- NLP (spaCy, NLTK)
- ML (scikit-learn)
- Transformers
- Streamlit
- Pandas
---
## **Workflow:**
1. Resume Upload
2. Preprocessing
3. Embedding Generation
4. Similarity Scoring
5. Ranking and Visualization
---
## **Installation:**
`pip install -r requirements.txt`
## **Usage:**
`streamlit run app.py`
## **Models Used:**
• <ins>Sentence Transformers</ins>
• Cosine Similarity
• TF-IDF Baseline (optional)
## **Example Output:**
• Rank
• Candidate Name
• Match Score
• Extracted Skills
• Missing Skills
## **Improvements:**
• LLM explanations
• Bias detection
• Feedback loop
• Online deployment
